Домашнее задание к уроку №19: “Flexbox”

**Цель:** Закрепить знания о Flexbox и научиться использовать его для выравнивания и управления расположением элементов на странице.

Задание 1: Свойство `display`

1. Создайте 3 `div`-элемента и примените к ним следующие значения свойства `display`:
   - Первый элемент — `block`
   - Второй элемент — `inline`
   - Третий элемент — `flex`

Понаблюдайте, как изменяется отображение элементов на странице при разных значениях `display`.

Задание 2: Flex-контейнер

1. Создайте `div`-контейнер с несколькими дочерними элементами.
2. Примените к контейнеру `display: flex` и посмотрите, как элементы внутри него автоматически выстраиваются в одну строку.

Задание 3: Свойства `flex-direction`, `flex-wrap`, `flex-flow`

1. Добавьте к flex-контейнеру из предыдущего задания следующее:
   - Примените `flex-direction: column`, чтобы расположить элементы в колонку.
   - Примените `flex-wrap`, чтобы элементы переносились на новую строку при нехватке места.
   - Используйте сокращённое свойство `flex-flow` для комбинирования направлений и обёртывания.

Задание 4: Выравнивание элементов

1. Используйте flex-контейнер и дочерние элементы. Примените свойства:
   - `justify-content` для горизонтального выравнивания элементов (например, по центру).
   - `align-items` для вертикального выравнивания элементов внутри контейнера.
   - `align-content` для управления выравниванием всех строк в flex-контейнере.

Задание 5: Свойства `flex-grow`, `flex-shrink`, `flex-basis` и сокращённое свойство `flex`

1. Создайте flex-контейнер с тремя элементами.
2. Примените к каждому элементу разные значения свойств `flex-grow`, `flex-shrink` и `flex-basis`, чтобы управлять их размерами.
3. Используйте сокращённое свойство `flex`, чтобы задать сразу все параметры для элемента.

Задание 6: Практическое задание

1. Создайте блок с тремя элементами внутри, используя flexbox. Расположите их так:
   - Первый элемент занимает 50% ширины контейнера.
   - Второй элемент занимает оставшуюся ширину и автоматически растягивается при увеличении размера контейнера.
   - Третий элемент всегда остаётся фиксированного размера (например, 100px).

Задание 7: Меню навигации с выравниванием по центру

Создайте горизонтальное меню навигации с использованием Flexbox, в котором ссылки будут выровнены по центру.

1. Создайте контейнер для меню и добавьте несколько ссылок (`<a>`).
2. Используйте Flexbox, чтобы выровнять ссылки по центру горизонтально и распределить их равномерно.
3. Реализуйте это меню и поэкспериментируйте с выравниванием ссылок с помощью свойств `justify-content` и `align-items`.

Задание 8: Галерея карточек

Создайте простую галерею карточек, используя Flexbox, где все карточки будут выровнены по сетке.

1. Создайте контейнер для галереи и несколько карточек внутри него.
2. Используйте Flexbox, чтобы разместить карточки в несколько колонок и равномерно распределить их по ширине контейнера.
3. Реализуйте галерею с несколькими карточками. Попробуйте изменить значение свойства `flex` и используйте свойства `flex-grow`, `flex-shrink`, и `flex-basis` для изменения размера карточек.
