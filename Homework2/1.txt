Домашнее задание к уроку №18: “Позиционирование элементов”

**Цель:** Закрепить понимание различных значений свойства `position` и их применение для создания различных макетов веб-страницы.

Задание 1: `position: static` (по умолчанию)

1. Создайте HTML-документ с несколькими блоками (`div`), содержащими текст.
2. Примените к ним CSS-стили по умолчанию, используя `position: static`.
3. Объясните, что по умолчанию элементы располагаются в обычном потоке документа.

Задание 2: `position: relative`

1. Создайте блок с текстом и примените к нему `position: relative`.
2. Используя свойства `top`, `right`, `bottom`, `left`, сместите блок на несколько пикселей относительно его обычного положения.
3. Объясните, как работает `position: relative`, показывая, что элемент остается на своем месте, но визуально смещается.

Задание 3: `position: absolute`

1. Создайте контейнерный блок и вложите в него несколько элементов.
2. Примените к одному из вложенных элементов `position: absolute` и с помощью свойств `top`, `right`, `bottom`, `left` расположите его внутри контейнера.
3. Покажите, как элемент позиционируется относительно ближайшего предка с `position: relative` или `absolute`.

Задание 4: `position: fixed`

1. Создайте блок с текстом и примените к нему `position: fixed`.
2. Расположите блок в правом нижнем углу экрана с помощью свойств `bottom` и `right`.
3. Прокрутите страницу вниз и убедитесь, что элемент остается на месте, несмотря на прокрутку.

Задание 5: `position: sticky`

1. Создайте несколько заголовков и абзацев на странице.
2. Примените к заголовкам `position: sticky` и установите значение `top`, чтобы заголовок "прилипал" к верхней части страницы при прокрутке.
3. Прокрутите страницу, чтобы увидеть эффект залипания заголовка.

Финальная страница

1. Соберите все задания в одном HTML-документе.
2. Для каждого примера добавьте комментарии, объясняющие работу различных значений свойства `position`.


**Требования:**
- Продемонстрируйте примеры с каждым значением свойства `position`: `static`, `relative`, `absolute`, `fixed`, `sticky`.
- Загрузить домашнее задание в GitHub и дать ссылку на репозиторий.
